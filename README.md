# analysis_filtering_stocks
상승율과 하락율이 큰 종목들을 추출하여 기업분석(재무제표) 및 기술적분석(차트분석)

# 실행환경
- python 3.10
- pyqt5

# 설명
### 엑셀 분석
1. 투자시점 분석
2. 종가에서 증가, 감소율이 높은 종목 추출
3. 증가감소비교분석
    - 증가감소비교_Form.xlsx
    - 증가내부비교_Form.xlsx
    - 감소내부비교_Form.xlsx
4. 필터링된 증가, 감소 종목 추출
5. 해당 종목의 차트 추출
